Veloo Public API v1

Welcome to the Veloo API. This is a read-only API for retrieving store and inventory data.

Authentication

All requests must include your API key in the request header:

X-API-Key: YOUR_TOKEN_HERE

Base URL

https://yourdomain.com/api/v1/

Endpoints

1. List Stores

GET /stores

Returns a list of all active stores.

Optional Query Params:
  • q — Search by store name
  • limit — Max results (default 50)
Example: 
curl -H "X-API-Key: YOUR_TOKEN_HERE" \
     https://yourdomain.com/api/v1/stores?q=gwarinpa

2. Store Inventory

GET /stores/{store_id}/items

Returns inventory items for the given store.

Optional Query Params:
  • q — Search by name or brand
  • in_stock — 1 or 0
  • low_stock_only — 1 to filter only low stock items
Example: 
curl -H "X-API-Key: YOUR_TOKEN_HERE" \
     https://yourdomain.com/api/v1/stores/12/items?in_stock=1

3. Single Item Details

GET /items/{item_id}

Returns details for one inventory item.

Example: 
curl -H "X-API-Key: YOUR_TOKEN_HERE" \
     https://yourdomain.com/api/v1/items/45

Response Format

All responses are in JSON.

{
    "success": true,
    "data": [
        { "id": 1, "name": "Product A", "price": 1200 }
    ]
}

Rate Limits

Default: 60 requests/minute per API key.

Errors

  • 401 Unauthorized — Missing/invalid API key
  • 404 Not Found — Resource doesn’t exist
  • 429 Too Many Requests — Rate limit exceeded
